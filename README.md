# HTTP-Server на чистом C

Минималистичный веб-сервер, написанный на C с использованием POSIX-сокетов. Обрабатывает GET-запросы, отдаёт статические файлы, поддерживает многопроцессную обработку через `fork()`. В комплекте - адаптивная страница портфолио.

## Что внутри
- Многопроцессная архитектура (один процесс на соединение)
- Раздача статики: HTML, CSS, изображения
- Защита от Directory Traversal (`..` в пути)
- Корректные MIME-типы (html, css, jpg, png)
- Логирование запросов в stdout
- Реакция на ошибки: 400, 404, 405

## Файлы
webserver-c/
+ webserver.c # исходник сервера
+ index.html # страница портфолио (отдаётся по умолчанию)
+ Makefile # правила сборки
+ README.md


## Сборка и запуск

### 1. Собрать
```bash
make
```
Или вручную:
```bash
gcc -Wall -Wextra -O2 -o webserver webserver.c
```
### 2. Запустить
```bash
./webserver
```
Сервер слушает порт 8080. В браузере:
```bash
http://localhost:8080
```
### 3. Остановить
`Ctrl+c` в терминале.

## Зависимости
+ GCC или Clang

+ POSIX-совместимая ОС (Linux, macOS, WSL)

+ Никаких внешних библиотек
